# GearGrid

ステージの高さ，幅はそれぞれ $H, W$

スタート，ゴールの座標はそれぞれ $sx, sy, gx, gy$

ステージの状態は $S[0], ... , S[H - 1]$で与えられる

回転壁の個数は $N$

各回転壁の情報は中心座標 $x[i], y[i]$

初期の壁の形 $shape[i]$

同期している回転壁の個数 $M[i]$

同期している回転壁の番号 $D[0], ... , D[M[i - 1]]$で与えられる

# 入力

$H \ W$

$sx \ sy$

$gx \ gy$

$S[0]$

$...$

$S[H - 1]$

$N$

$x[0] \ y[0] \ shape[0] \ M[0] \ D[0] \ ... \ D[M[0]]$

$x[N - 1] \ y[N - 1] \ shape[N - 1] \ M[N - 1] \ D[N - 1] \ ... \ D[M[N - 1]]$
